The revised model

In this version of the model, Russia is a price taker that solves

\[ \begin{eqnarray*} \mbox{ Maximize }\sum_{t=1}^{T}\beta^{t-1} \{ p_B K_{t}&+&\hat{p}Q_{t}-C(K_{t}+Q_{t})-F(I_{t})\}\\ % &&\\ K_{1}&=&\bar{k}>0\\ K_{t+1}&=&K_{t}+I_{t}\\ K_{t}, I_{t},Q_{t}& \geq &0 \end{eqnarray*} \]

  • Choice variables:
    • \(\left\{Q_t\right\}_{t=1}^{T}\): Quantity supplied through Western channels
    • \(\left\{K_t\right\}_{t=1}^{T}\): Shadow fleet capacity = quantity supplied through non-Western channels
    • \(\left\{I_t\right\}_{t=1}^{T}\): Investment in the development of non-Western channel capacity
  • Parameters:
    • \(\beta\): Discount rate
    • \(\hat{p}\): Cap price on oil sold through Western channels
    • \(d \ge 0\): discount under with Russia can sell crude at Non-Western channels
    • \(p_{B}(.)\): (Inverse) Global oil demand
    • \(K_1\): Initial capacity of non-Western channels
    • \(K_a\): The first breakpoint in capacity corresponding to the global price
    • \(Z(p)\): Non-russian (rest of the world) oil supply (assumed constant)
    • \(C(\cdot)\): Cost function of production
    • \(F(\cdot)\): Cost function of capacity investment

Global demand function

We will follow Gars et al (2022) and Wachtmeister et al (2023) and specify an isoelastic demand function from the baseline price and global demand

\[ D_{W}(p_B) = D_0 * \left(\frac{p_B}{p_0}\right)^{-\epsilon_D} \]

where \(D_0\) and \(p_0\) are the baseline (pre-war) price and demand and \(\epsilon_D\) is the absolute value of the price elasticity of demand. This gives the inverse demand function

\[ p_{B}(D_{t}) = \alpha_D D_t^{-1/\epsilon_{D}} \]

where \(\alpha_D \equiv p_0 {D_0}^{1/\epsilon_D}\) is a constant defined by the baseline. The computational version does not calculate \(\alpha_D\) separately due to pontential scaling problems (its in the order of \(10^33\) with the current units).

Code 
p_0 = 80 # Baseline price (mUSD/mb)
D_0 = S_ROW_0 + Q_W_0 + Q_NW_0 # Combining IEA data from the Russian report and that used in Gars et al
ϵ_D = 0.125 # Short-run case from Gars et al
p_B(D_t) = p_0 * (D_t/D_0)^(-1/ϵ_D)

Production cost function

Using Rystad data from 2020, we can observe that the long-run marginal cost curve (built with the same methods from Julien, Saraly, and Diego’s WP) looks rather flat for a bigger portion.

On a separate script, I found that the portion between 2 and 42 billion barrels has a great fit with the linear equation \(y = 19.42 + 0.92x\). The problem with this approach, however, is that this is a long-run marginal cost curve and we need a monthly production cost curve.

Also based on Rystad data, Gars et al. (2022) find a baseline cost of production for Russia at $17/bbl.

In constrast, Watchmeister et al. (2023) have the following curve

Eyeballing their data, we can also observe approximatelly linear marginal cost functions, but with a much lower intercept.

For a parsimonius specification, we will specify a linear marginal cost function with intercept \(c_0 = 17\) and slope \(\bar{c}\).

Based on the IEA data, roughly calibrating it for Jan. 2022, would give us

\[ \begin{eqnarray} p_0 = \frac{dC}{dQ} = c_0 + \bar{c} Q_0\\ \bar{c} = \frac{p_0 - c_0}{Q_0} \\ \bar{c} \approx 0.095 \end{eqnarray} \]

where \(\theta_0\) is Russia’s initial market share. A low slope is very much in line with the papers cited above. For this one, I used the long-run elasticity.

This gives us the following function. (If the solution happens to go beyond the level of production in the linear part, I will include the hockey stick part).

Code

c_0 = 17.0
c̄ = (p_0 - c_0)/(Q_NW_0 + Q_W_0)

C(Q) = c_0*Q + c̄*Q^2/2

Critical points

These functional forms allow us to determine the location of two critical points in \(K\). The first one is

\[ K_a = c^{\prime-1}\left(\hat{p}\right) = \frac{\hat{p}-c_{0}}{\bar{c}} \]

Code 
K_a = (p̂ - c_0)/c̄
454.57142857142856

With \(\hat{p} = 60\), \(K_a \approx 455\) mb/q or 5 mb/d. This can be interpreted as the maximum value of \(K\) such that \(Q \ge 0\); any capacity expansion beyond that will take \(Q\) to the corner solution.

Another critical point for \(K\) is given when the capacity constraint stops binding \[ K_{b}\text{ s.t. }p_{B}\left(Z+K_{b}\right)=c^{\prime}\left(K_{b}\right)\Rightarrow K_{b}\approx 666 \mbox{ mb/q or } 7.4 \mbox{ mb/d} \]

Which we can easily solve using rootfinding

Code 
eqn(k) = p_B(Z + k[1]) - c_0 - c̄*k[1]
nl_res = nlsolve(eqn, [1.0])
K_b = nl_res.zero[1]

Investment cost function

For simplicity, I will also define quadratic cost functions for capacity investments with marginals costs having constant slope \(\bar{f}\) and zero intercept.

The capacity expansion cost function is harder to calibrate upfront because of its dynamic component. A simple heuristc can be derived by the fact that \(\frac{dF}{dI_t} = \beta \frac{\partial V}{\partial I_t}\). One additional unit of capacity will allow Russia to sell one unit in all subsequent periods, which might bring additional revenue. Initially, the additional revenue will be \(p_0 - \hat{p}\), but as capacity expands, this margin decreases.

This version will assume a very simple path to have preliminary results. I will revise it to incorporate function parameter estimation along with the numerical solution. I will assume that Russia made the decision on the first month as if periodic revenue benefits decline with a decay factor \(\omega\) over 80 periods (40 years). This is, the NPV of added revenue in period 1 is

\[ \frac{\partial V}{\partial I_1} = (p_B - \hat{p})\frac{1 - (\beta\omega)^{80}}{1-\beta\omega} \]

Assuming that in the period observed, quarterly capacity added was on average \(\Delta K/8\), we can write

\[ f^\prime(\frac{K_{2023}-K_{2021}}{8}) = \frac{468-180}{8} \bar{f} = (p_0 - \hat{p})\beta\frac{1 - (\beta\omega)^{80}}{1-\beta\omega} \Rightarrow \bar{f} \approx 9.1 \]

Code 
# Define cost function F(I)
f̄ = (p_0 - p̂)*β*(1 - (β*ω)^80)/(1-β*ω) / ((468-180)/8)

F(I) = f̄ * I^2/2
F (generic function with 1 method)

Solution strategy 1: Nonlinear system of equations

Given that time is discrete and finite, we can solve the system of equations implied by the equilibrium conditions

\[ \begin{eqnarray} \mbox{transition: } & k_{t+1}&=k_{t}+I_{t}\\ \mbox{cap sales: } &Q_{t}&\geq 0, \hat{p}-C'(k_{t}+Q_{t}) \leq 0, c.s.\\ \mbox{investment: }&F'(I_{t})&=\beta\{F'(I_{t+1})+ p(z+\max(k_{a},k_{t}+I_{t})) - d - C'(k_{t}+I_{t}+Q_{t+1})\}\\ \mbox{initial condx: }&k_{1}&=\bar{k}\\ \mbox{terminal condx: }&F'(I_{T-1})&=\beta\{p(z+\max(k_{a},k_{T-1}+I_{T-1}))-C'(k_{T-1}+I_{T-1}+Q_{T})\}. \end{eqnarray} \]

Baseline case (\(d = 0\))

We start with the case where there is no discount to compare with previous solution methods.

Code 
mod1 = Model(Ipopt.Optimizer)

# Variable definitions, box constraints, and initial values
@variable(mod1, I[1:T] >= 0)
@variable(mod1, k[1:T] >= 0, start = K₁)
@variable(mod1, Q[1:T] >= 0, start = Q_W_0)

# Set d, cap, and K_1 as parameters so we can reuse the model
@variable(mod1, this_d in Parameter(1))
@variable(mod1, this_cap in Parameter(1))
@variable(mod1, K_initial in Parameter(1))

# Constraints
# Transition
@constraint(mod1, [t in 2:T], k[t] == k[t-1] + I[t-1])

# Cap sales
@constraint(mod1, [t in 1:T], Q[t] * (this_cap - C_prime(k[t] + Q[t])) == 0)

# Investment
@constraint(mod1, [t in 1:(T-2)], F_prime(I[t]) == β*(F_prime(I[t+1]) + p_B(Z + maximum([K_a, k[t+1]])) - this_d -C_prime(k[t+1] + Q[t+1])))

# Initial
@constraint(mod1, k[1] == K_initial)

# Terminal
@constraint(mod1, F_prime(I[T-1]) == β*(p_B(Z + maximum([K_a, k[T]])) -this_d -C_prime(k[T] + Q[T])))
@constraint(mod1, I[T] == 0)

@objective(mod1, Max, 0) # Dummy objective (just need to ensure constraints bind)
0

Function to assemble paths and calculate outcomes.

Code 
function calculate_path(K, Q, I, disc, cap, K_initial)
  # Pre-allocate a DataFrame 
  traj = DataFrame(
    t_grid = collect(0:T),
    K  = vcat(K_initial, K),
    Q  = vcat(Q_W_0, Q),
    I  = vcat(0.0, I)
  )

  traj.P  = p_B.(traj.K .+ traj.Q .+ S_ROW_0)
  
  traj.CS = CS.(traj.K, traj.Q, disc, cap)
  traj.CS[1] = CS(traj.K[1], traj.Q[1], 0.0, traj.P[1])
  
  traj.Prof = Profit.(traj.P, traj.K, traj.Q, traj.I, disc, cap)
  traj.Prof[1] = Profit(traj.P[1], traj.K[1], traj.Q[1], traj.I[1], 0.0, traj.P[1])
  return traj
end;

Function to assemble a collection of trajectories for later plotting.

Code 
function solve_models(model, disc_values, cap, K_1)
  set_silent(model);
  solution_dfs = Vector{DataFrame}();
  
  for d in disc_values
    set_parameter_value(this_d, d);
    set_parameter_value(this_cap, cap);
    set_parameter_value(K_initial, K_1)
    optimize!(mod1);
    push!(solution_dfs, calculate_path(value.(k), value.(Q), value.(I), d, cap, K_1));
  end
  return solution_dfs;
end

## The revised model

In this version of the model, Russia is a price taker that solves

$$
\begin{eqnarray*}
\mbox{ Maximize }\sum_{t=1}^{T}\beta^{t-1} \{ p_B K_{t}&+&\hat{p}Q_{t}-C(K_{t}+Q_{t})-F(I_{t})\}\\
% &&\\
K_{1}&=&\bar{k}>0\\
K_{t+1}&=&K_{t}+I_{t}\\
K_{t}, I_{t},Q_{t}& \geq &0
\end{eqnarray*}
$$

- Choice variables:
  -  $\left\{Q_t\right\}_{t=1}^{T}$: Quantity supplied through Western channels
  -  $\left\{K_t\right\}_{t=1}^{T}$: Shadow fleet capacity = quantity supplied through non-Western channels
  -  $\left\{I_t\right\}_{t=1}^{T}$: Investment in the development of non-Western channel capacity

- Parameters:
  - $\beta$: Discount rate
  - $\hat{p}$: Cap price on oil sold through Western channels
  - $d \ge 0$: discount under with Russia can sell crude at Non-Western channels
  - $p_{B}(.)$: (Inverse) Global oil demand
  - $K_1$: Initial capacity of non-Western channels
  - $K_a$: The first breakpoint in capacity corresponding to the global price
  - $Z(p)$: Non-russian (rest of the world) oil supply (assumed constant)
  - $C(\cdot)$: Cost function of production
  - $F(\cdot)$: Cost function of capacity investment

**Global demand function**

We will follow Gars et al (2022) and Wachtmeister et al (2023) and specify an isoelastic demand function from the baseline price and global demand

$$
D_{W}(p_B) = D_0 * \left(\frac{p_B}{p_0}\right)^{-\epsilon_D}
$$

where $D_0$ and $p_0$ are the baseline (pre-war) price and demand and $\epsilon_D$ is the absolute value of the price elasticity of demand. This gives the inverse demand function

$$
p_{B}(D_{t}) = \alpha_D D_t^{-1/\epsilon_{D}} 
$$

where $\alpha_D \equiv p_0 {D_0}^{1/\epsilon_D}$ is a constant defined by the baseline. The computational version does not calculate $\alpha_D$ separately due to pontential scaling problems (its in the order of $10^33$ with the current units).

```{julia}
p_0 = 80 # Baseline price (mUSD/mb)
D_0 = S_ROW_0 + Q_W_0 + Q_NW_0 # Combining IEA data from the Russian report and that used in Gars et al
ϵ_D = 0.125 # Short-run case from Gars et al
p_B(D_t) = p_0 * (D_t/D_0)^(-1/ϵ_D)
```

**Production cost function**

Using Rystad data from 2020, we can observe that the long-run marginal cost curve (built with the same methods from Julien, Saraly, and Diego's WP) looks rather flat for a bigger portion.

![](./../../../output/figs/Russia_LRMC_2020.png)

On a separate script, I found that the portion between 2 and 42 billion barrels has a great fit with the linear equation $y = 19.42 + 0.92x$. The problem with this approach, however, is that this is a long-run marginal cost curve and we need a monthly production cost curve.

Also based on Rystad data, Gars et al. (2022) find a baseline cost of production for Russia at \$17/bbl.

In constrast, Watchmeister et al. (2023) have the following curve

![](./../../../data/figs/Wachtmeister_Fig_1_MC.png)

Eyeballing their data, we can also observe approximatelly linear marginal cost functions, but with a much lower intercept. 

For a parsimonius specification, we will specify a linear marginal cost function with intercept $c_0 = 17$ and slope $\bar{c}$.  

Based on the IEA data, roughly calibrating it for Jan. 2022, would give us 

$$
\begin{eqnarray}
p_0 = \frac{dC}{dQ} = c_0 + \bar{c} Q_0\\
\bar{c} = \frac{p_0 - c_0}{Q_0}  \\
\bar{c} \approx 0.095
\end{eqnarray}
$$

where $\theta_0$ is Russia's initial market share. A low slope is very much in line with the papers cited above. For this one, I used the long-run elasticity.

This gives us the following function. (If the solution happens to go beyond the level of production in the linear part, I will include the hockey stick part).

```{julia}
# Define cost function C(Q)
c_0 = 17.0
c̄ = (p_0 - c_0)/(Q_NW_0 + Q_W_0)

C(Q) = c_0*Q + c̄*Q^2/2
``` 


For the new solution strategy, it will be useful to have the first derivative programmed

```{julia}
C_prime(Q) = c_0 + c̄*Q
```

**Critical points**

These functional forms allow us to determine the location of two critical points in $K$. The first one is


$$
K_a = c^{\prime-1}\left(\hat{p}\right) = \frac{\hat{p}-c_{0}}{\bar{c}}
$$

```{julia}
K_a = (p̂ - c_0)/c̄
```

With $\hat{p} = 60$, $K_a \approx 455$ mb/q or 5 mb/d. This can be interpreted as the maximum value of $K$ such that $Q \ge 0$; any capacity expansion beyond that will take $Q$ to the corner solution.

Another critical point for $K$ is given when the capacity constraint stops binding
$$
K_{b}\text{ s.t. }p_{B}\left(Z+K_{b}\right)=c^{\prime}\left(K_{b}\right)\Rightarrow K_{b}\approx 666 \mbox{ mb/q or } 7.4 \mbox{ mb/d}
$$

Which we can easily solve using rootfinding

```{julia}
eqn(k) = p_B(Z + k[1]) - c_0 - c̄*k[1]
nl_res = nlsolve(eqn, [1.0])
K_b = nl_res.zero[1] 
```

**Investment cost function**

For simplicity, I will also define quadratic cost functions for capacity investments with marginals costs having constant slope $\bar{f}$ and zero intercept.

The capacity expansion cost function is harder to calibrate upfront because of its dynamic component. A simple heuristc can be derived by the fact that $\frac{dF}{dI_t} = \beta \frac{\partial V}{\partial I_t}$. One additional unit of capacity will allow Russia to sell one unit in all subsequent periods, which might bring additional revenue. Initially, the additional revenue will be $p_0 - \hat{p}$, but as capacity expands, this margin decreases.

This version will assume a very simple path to have preliminary results. I will revise it to incorporate function parameter estimation along with the numerical solution. I will assume that Russia made the decision on the first month as if periodic revenue benefits decline with a decay factor $\omega$ over 80 periods (40 years). This is, the NPV of added revenue in period 1 is

$$
\frac{\partial V}{\partial I_1} = (p_B - \hat{p})\frac{1 - (\beta\omega)^{80}}{1-\beta\omega}
$$

Assuming that in the period observed, quarterly capacity added was on average $\Delta K/8$, we can write

$$
f^\prime(\frac{K_{2023}-K_{2021}}{8}) = \frac{468-180}{8} \bar{f} = (p_0 - \hat{p})\beta\frac{1 - (\beta\omega)^{80}}{1-\beta\omega} \Rightarrow \bar{f} \approx 9.1
$$

```{julia}
# Define cost function F(I)
f̄ = (p_0 - p̂)*β*(1 - (β*ω)^80)/(1-β*ω) / ((468-180)/8)

F(I) = f̄ * I^2/2
```

For the new solution strategy, it will be useful to have the first derivative programmed

```{julia}
F_prime(I) = f̄ * I
```

### Solution strategy 1: Nonlinear system of equations

Given that time is discrete and finite, we can solve the system of equations implied by the equilibrium conditions

$$
\begin{eqnarray}
    \mbox{transition: } & k_{t+1}&=k_{t}+I_{t}\\
    \mbox{cap sales: } &Q_{t}&\geq 0, \hat{p}-C'(k_{t}+Q_{t}) \leq 0, c.s.\\
    \mbox{investment: }&F'(I_{t})&=\beta\{F'(I_{t+1})+ p(z+\max(k_{a},k_{t}+I_{t})) - d - C'(k_{t}+I_{t}+Q_{t+1})\}\\
    \mbox{initial condx: }&k_{1}&=\bar{k}\\
    \mbox{terminal condx: }&F'(I_{T-1})&=\beta\{p(z+\max(k_{a},k_{T-1}+I_{T-1}))-C'(k_{T-1}+I_{T-1}+Q_{T})\}.
\end{eqnarray}
$$


#### Baseline case ($d = 0$)

We start with the case where there is no discount to compare with previous solution methods.

```{julia}
mod1 = Model(Ipopt.Optimizer)

# Variable definitions, box constraints, and initial values
@variable(mod1, I[1:T] >= 0)
@variable(mod1, k[1:T] >= 0, start = K₁)
@variable(mod1, Q[1:T] >= 0, start = Q_W_0)

# Set d, cap, and K_1 as parameters so we can reuse the model
@variable(mod1, this_d in Parameter(1))
@variable(mod1, this_cap in Parameter(1))
@variable(mod1, K_initial in Parameter(1))

# Constraints
# Transition
@constraint(mod1, [t in 2:T], k[t] == k[t-1] + I[t-1])

# Cap sales
@constraint(mod1, [t in 1:T], Q[t] * (this_cap - C_prime(k[t] + Q[t])) == 0)

# Investment
@constraint(mod1, [t in 1:(T-2)], F_prime(I[t]) == β*(F_prime(I[t+1]) + p_B(Z + maximum([K_a, k[t+1]])) - this_d -C_prime(k[t+1] + Q[t+1])))

# Initial
@constraint(mod1, k[1] == K_initial)

# Terminal
@constraint(mod1, F_prime(I[T-1]) == β*(p_B(Z + maximum([K_a, k[T]])) -this_d -C_prime(k[T] + Q[T])))
@constraint(mod1, I[T] == 0)

@objective(mod1, Max, 0) # Dummy objective (just need to ensure constraints bind)
```

Function to calculate outcomes (CS, profits, etc)

```{julia}
CS(K, Q, disc, cap) = ϵ_D/(1-ϵ_D)*(Z*p_B(Z)-(Z+K+Q)*p_B(Z+K+Q)) - K * (p_B(Z+K) - disc) - Q * cap 
Profit(P, K, Q, I, disc, cap) = (P - disc) * K + cap * Q - C(K + Q) - F(I)
```

Function to assemble paths and calculate outcomes.

```{julia}
function calculate_path(K, Q, I, disc, cap, K_initial)
  # Pre-allocate a DataFrame 
  traj = DataFrame(
    t_grid = collect(0:T),
    K  = vcat(K_initial, K),
    Q  = vcat(Q_W_0, Q),
    I  = vcat(0.0, I)
  )

  traj.P  = p_B.(traj.K .+ traj.Q .+ S_ROW_0)
  
  traj.CS = CS.(traj.K, traj.Q, disc, cap)
  traj.CS[1] = CS(traj.K[1], traj.Q[1], 0.0, traj.P[1])
  
  traj.Prof = Profit.(traj.P, traj.K, traj.Q, traj.I, disc, cap)
  traj.Prof[1] = Profit(traj.P[1], traj.K[1], traj.Q[1], traj.I[1], 0.0, traj.P[1])
  return traj
end;
```

#### Welfare results


We then calculate total NPV of Russia's profits and CS over all periods (20 years) for comparisons in each case and scenario.

```{julia}
function calculate_outcomes(traj)
  pt_traj = traj[2:nrow(traj), :] # Skip pre-sanction period
  profits = sum(β.^(pt_traj.t_grid .- 1) .* pt_traj.Prof)/1000
  cs = sum(β.^(pt_traj.t_grid .- 1) .* pt_traj.CS)/1000
  return profits, cs
end;
```

```{julia}
BAU_Prof, BAU_CS = calculate_outcomes(traj_bau)
```

```{julia}
w_df_BAU = DataFrame(
  scenario = ["BAU"],
  d = 0.0,

  CS   = [BAU_CS]
  );
```
